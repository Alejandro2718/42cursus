# üìöFunctions Explanation:

## [ft_isalpha](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_isalpha.c)
Checks if a character is an alphabetic letter (a-z, A-Z)

## [ft_isdigit](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_isdigit.c)
Checks if a character is a digit (0-9)

## [ft_isalnum](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_isalnum.c)
Checks if a character is alphanumeric (a-z, A-Z, 0-9)

## [ft_isascii](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_isascii.c)
Checks if a character is a valid ASCII character (0-127)

## [ft_isprint](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_isprint.c)
Checks if a character is printable (including space)

## [ft_strlen](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_strlen.c)
Calculates the length of a string (number of characters before the null terminator)

## [ft_memset](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_memset.c)
Fills a block of memory with a specific value

## [ft_bzero](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_bzero.c)
Sets a block of memory to zero

## [ft_memcpy](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_memcpy.c)
Copies a block of memory from one location to another

## [ft_memmove](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_memmove.c)
Copies a block of memory, handling overlapping regions safely

## [ft_strlcpy](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_strlcpy.c)
Copies a string to a destination with a specified size, ensuring null-termination

## [ft_strlcat](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_strlcat.c)
Concatenates two strings with a specified size, ensuring null-termination

## [ft_toupper](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_toupper.c)
Converts a lowercase letter to uppercase

## [ft_tolower](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_tolower.c)
Converts an uppercase letter to lowercase

## [ft_strchr](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_strchr.c)
Finds the first occurrence of a character in a string

## [ft_strrchr](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_strrchr.c)
Finds the last occurrence of a character in a string

## [ft_strncmp](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_strncmp.c)
Compares up to a specified number of characters of two strings

## [ft_memchr](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_memchr.c)
Searches for the first occurrence of a character in a block of memory

## [ft_memcmp](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_memcmp.c)
Compares two blocks of memory

## [ft_strnstr](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_strnstr.c)
Locates the first occurrence of a substring in a string, up to a specified length

## [ft_atoi](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_atoi.c)
Converts a string to an integer

## [ft_calloc](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_calloc.c)
Allocates memory for an array of elements, initializes the memory to zero, and returns a pointer to it. It takes the number of elements and the size of each element as arguments

## [ft_strdup](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_strdup.c)
Duplicates a string by allocating memory for a copy of the string and copying the contents of the original string into the newly allocated memory. It returns a pointer to the duplicated string

## [ft_substr](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_substr.c)
Creates a substring from a given string. It takes the original string, a starting index, and a length, and returns a new string containing the specified portion of the original string

## [ft_strjoin](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_strjoin.c)
Concatenates two strings into a new string. It allocates memory for the resulting string and returns a pointer to it

## [ft_strtrim](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_strtrim.c)
Removes specified characters (by default, whitespace) from the beginning and end of a string, returning a new trimmed string

## [ft_split](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_split.c)
Splits a string into an array of substrings based on a specified delimiter. It returns a null-terminated array of strings

## [ft_itoa](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_itoa.c)
Converts an integer to a string. It allocates memory for the resulting string and returns a pointer to it

## [ft_strmapi](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_strmapi.c)
Applies a function to each character of a string, creating a new string with the results. It takes the original string and a function as arguments

## [ft_striteri](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_striteri.c)
Applies a function to each character of a string, modifying the string in place. The function receives the index of the character and the character itself

## [ft_putchar_fd](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_putchar_fd.c)
Writes a single character to a specified file descriptor

## [ft_putstr_fd](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_putstr_fd.c)
Writes a string to a specified file descriptor

## [ft_putendl_fd](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_putendl_fd.c)
Writes a string to a specified file descriptor, followed by a newline character

## [ft_putnbr_fd](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_putnbr_fd.c)
Writes an integer to a specified file descriptor

# ‚≠êBONUS Functions:
 The following are functions to manipulating lists <- (data structure)

## [ft_lstnew](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_lstnew.c)
Creates a new list node (element) for a linked list. It allocates memory for the node and initializes its content and next pointer

## [ft_lstadd_front](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_lstadd_front.c)
Adds a new node to the front (beginning) of a linked list. It updates the head of the list to point to the new node

## [ft_lstsize](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_lstsize.c)
Counts the number of nodes in a linked list and returns the size (length) of the list

## [ft_lstlast](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_lstlast.c)
Finds and returns the last node in a linked list. If the list is empty, it returns NULL

## [ft_lstadd_back](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_lstadd_back.c)
Adds a new node to the end (back) of a linked list. If the list is empty, the new node becomes the head of the list

## [ft_lstdelone](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_lstdelone.c)
Deletes a single node from a linked list. It takes a node and a deletion function to free the node's content, then frees the node itself

## [ft_lstclear](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_lstclear.c)
Deletes and frees all nodes in a linked list. It uses a deletion function to free each node's content and frees the nodes themselves

## [ft_lstiter](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_lstiter.c)
Iterates over a linked list and applies a function to the content of each node

## [ft_lstmap](https://github.com/Alejandro2718/42cursus/blob/main/Libft/ft_lstmap.c)
Creates a new linked list by applying a function to the content of each node in the original list. It returns the new list
